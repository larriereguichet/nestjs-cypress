# nestjs-cypress

This is a minimum reproducible example attempting to run **Cypress** e2e tests against a **nestjs** api
which uses a `dto` annotated with an `@ApiProperty()` decorator from `@nestjs/swagger`.

```
npm install
npm run test:cypress
```
